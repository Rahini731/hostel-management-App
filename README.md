# Hostel Management App

Simple Expo React Native frontend for a hostel management application.

## Structure

- `hostel_app/` — Expo React Native application source files.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the app (Expo):

```bash
npm start
```

## Notes

- This repository currently contains only the frontend (Expo) app. No backend is included.
- CI: basic GitHub Actions workflow runs `npm ci` and any `build`/`test` scripts if present.
